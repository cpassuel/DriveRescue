Sélection du disque dur

Sélection disque physique

Avant de pouvoir travailler sur le disque dur, il faut le sélectionner. Pour cela, allez dans le menu Drives/Select Drive... pour ouvrir la fenêtre de sélection de disques.



Sélectionner un disque dans la liste et cliquez sur OK. Un message de confirmation va s'inscrire dans la zone de dialogue, et le modèle du disque va s'afficher dans la barre de titre.


Note: Sous Windows 9x, si le nom du modèle n'est pas détaillé, c'est peut être parce que le fichier SMARTVSD.VXD n'est pas installé correctement ; voir ici pour plus d'information.


Sélection disque virtuel

Non implémenté

Ce mode permet de travailler sur l'image d'un disque stocké sur fichier.

Informations sur le disque

Une fois le disque dur sélectionné, il est possible de voir ses caractéristiques en sélectionnant le menu Drive/Drive Info... Une fenêtre indiquant les propriétés du disque va s'afficher.


Il est possible de sauvegarder ces informations dans un fichier texte en cliquant sur le bouton Save.

Visualisation des secteurs

Il est possible de visualiser le contenu des secteurs du disque avec le menu Drives/View Drive Sectors. La sauvegarde des informations sur fichier peut se faire en format hexa (tel qu'il est présenté à l'écran) ou binaire (copie exacte du secteur).

Liste des partitions

La fonctions Drive/Partition List Dump permet d'afficher dans la zone message, la liste des partitions sur le disque sélectionné même si celles-ci ne sont pas supportées dans la partie analyse. Cela permet de faire un diagnostic rapide sur les partitions trouvées (partition cachées, système de fichiers spécifique, etc.)


Par exemple :

Info partition n° 1

 - FileSystem: FAT 32 (BootID: 0B)
 - Status : Logical
 - Start Sector: 24579513 (CHS 1530-1-1)
 - Nombre de secteurs: 22025052
 - Taille de la partition: 10,5 Go

Il être intéressant de sauvegarder ces informations sur un support externe et fiable après un partitionnement et formatage du disque, car ces informations sont utiles pour une récupération ultérieure d'une partition corrompue. Pour plus d'informations sur une partition, il faut la sélectionner et afficher les informations comme indiqué ici.
